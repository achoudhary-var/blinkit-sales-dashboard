# 🛒 Blinkit Sales Dashboard

## 📌 Problem Statement
Analyze Blinkit sales data to understand revenue trends and product performance.

## 🎯 Objective
To build an interactive dashboard for tracking sales KPIs and identifying insights.

## 🛠 Tools Used
- Microsoft Excel

## 📊 Dashboard Features
- Total Sales KPI
- Category-wise performance
- Monthly trends
- Top products analysis

## 📈 Key Insights
- Certain categories contribute majority of revenue
- Seasonal trends observed
- High-performing products identified
